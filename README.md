# Game: BoilingPoint | Игра: Точка кипения

> Русский

### Мини-игра
  Мини-игра "Точка кипения" представляет собой инновационный проект, реализованный в виде консольного приложения. Данный продукт демонстрирует применение передовых алгоритмов искусственного интеллекта для создания интерактивных игровых сценариев.

Основной концепт игры заключается в следующем: после инициализации пользователь получает возможность настроить параметры сложности, количество попыток, предоставление подсказок и другие ключевые характеристики. В процессе игры бот генерирует случайное число, что требует от пользователя его определения с учетом установленных параметров.

Ключевой особенностью игры "Точка кипения" является адаптивное поведение программы. В зависимости от текущих результатов пользователя, система выполняет определенные действия, направленные на оптимизацию игрового процесса. Это может включать как упрощение задачи для поддержания интереса игрока, так и ее усложнение для обеспечения достаточного уровня вызова и интеллектуального удовлетворения.

### Mini-game
  The "BoilingPoint" mini-game is an innovative project implemented as a console application. This product demonstrates the use of advanced artificial intelligence algorithms to create interactive game scenarios.

The basic concept of the game is as follows: after initialization, the user gets the opportunity to adjust the difficulty settings, the number of attempts, the provision of hints and other key characteristics. During the game, the bot generates a random number, which requires the user to determine it based on the set parameters.

The key feature of "BoilingPoint" is the adaptive behavior of the program. Depending on the user's current results, the system performs certain actions aimed at optimizing the gameplay. This may include simplifying the task to keep the player interested, or making it more difficult to ensure a sufficient level of challenge and intellectual satisfaction.
